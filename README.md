# 💻 Portofolio Damar Adin Firdaus - V3

Website portofolio pribadi versi ke-3 yang dirancang untuk menampilkan profil, keahlian, pengalaman, dan proyek-proyek pengembangan web. Dibangun dengan antarmuka modern bertema *Cyberpunk/Neon Dark Mode*, animasi interaktif, dan performa yang sangat dioptimalkan.

🌐 **[Lihat Live Demo di Sini] (Ganti_dengan_Link_Website_Kamu_Nantinya)**

---

## ✨ Fitur Utama

- **Desain Modern & Responsif:** Tampilan UI/UX yang disesuaikan dengan sempurna untuk semua perangkat (Desktop, Tablet, dan Mobile/HP).
- **Tema Cyberpunk/Neon:** Penggunaan warna gelap (Void/Deep) dengan aksen *Cyan Glow* dan *Blue Neon* yang futuristik.
- **Background Partikel Interaktif:** Animasi partikel *node* yang saling terhubung menggunakan HTML5 Canvas.
- **Efek Mengetik Dinamis (Typing Effect):** Animasi teks berjalan pada bagian Hero Section.
- **Pop-up Sertifikat (Modal/Lightbox):** Sistem interaktif untuk melihat gambar sertifikat tanpa berpindah halaman.
- **Formulir Kontak Aktif:** Formulir yang langsung terhubung ke email menggunakan integrasi API Web3Forms.
- **Dukungan Multi-bahasa:** Terintegrasi dengan Google Translate untuk aksesibilitas global.
- **Scroll Reveal & Back-to-Top:** Animasi halus saat pengguna menggulir halaman ke bawah, dilengkapi tombol pintas untuk kembali ke atas.

---

## 🛠️ Teknologi yang Digunakan (Tech Stack)

Proyek ini dibangun sepenuhnya dari awal (Native) tanpa menggunakan *framework* berat untuk menjaga performa tetap cepat:

- **HTML5:** Struktur semantik.
- **CSS3:** Custom Properties (Variables), Flexbox, CSS Grid, Media Queries, dan Animasi *Keyframes*.
- **Vanilla JavaScript (ES6+):** DOM Manipulation, HTML5 Canvas API, dan Intersection Observer API (untuk *scroll reveal*).
